Admin · Task

Give admins a path they can trust.

Prompts & LLM · Situation

Q&A worked — buying didn’t stop there.

What was still missing

People still needed catalog lines, quantities, real fields, pricing — not only policy answers and a link away.

If we “fixed” that with more free-form model text, we’d get confident answers that don’t match what’s on screen. The UI and the prompts have to stay in lockstep.

Employee needSo we had to…
Pick & submit catalog itemsKeep it in the same thread, step by step
Off-catalog requestsReal fields inline — no surprise jump-out
PricingSame conversation, same policy story
Plain story
Longer flows are still the same assistant — not a pile of new tricks.

Prompts & LLM · Task

Finish the purchase without breaking the habit.

Same column · Catalog, off-catalog, and pricing all happen where people already trust the assistant from Phase 1.

Same beat · Each step stays answer → where it came from → one clear next thing — not five competing buttons from the model.

Same admin rules · What admins loaded and routed still feeds what the model can do in these longer flows.
